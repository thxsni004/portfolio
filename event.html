Event Management Website

Project Name: Moment Makers

This is a full-stack event management platform built using the MERN stack (MongoDB, Express.js, React.js, Node.js). The application helps users plan and manage events with ease, including wedding planning, stage shows, food packages, seating arrangements, and more.

Key Features

  • Customizable event categories for weddings, stage shows, and more
  • Real-time order management and booking history
  • Admin panel for managing events and prices
  • Payment integration with Razorpay (dummy mode)
  • User profile to manage order details and settings

Technologies Used

  • MongoDB
  • Express.js
  • React.js
  • Node.js
  • Razorpay (for payment processing)
  • JavaScript (ES6+)
