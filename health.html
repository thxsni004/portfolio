Health Tracker Website

Project Name: Health Tracker

This is a full-stack health tracking platform built using the MERN stack (MongoDB, Express.js, React.js, Node.js). The application helps users monitor and improve their health by tracking daily habits such as exercise, meals, sleep, water intake, and steps.

Key Features

  • Daily tracking of exercise, meals, water, sleep, and steps
  • Interactive dashboard with graphs and personalized health tips
  • Reminder system for water and step goals

Technologies Used

  • MongoDB
  • Express.js
  • Node.js
  • JavaScript (ES6+)
